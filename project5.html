"Meme game" project

That project was made for practicing some CSS fetures and events. It is a game where you have a phrase and four pictures to react to that phrase. You can choose an appropriate meme picture and then drag and drop it to the dashed square. Each answer is counted and in the end your score is shown. I used HTML base file, CSS files and JavaScript file with functions to manipulate with elements of the DOM.
